# ventures-pm worker

**Mission:** Manage client projects (SyneticX, Halalbooking, PriPitch, Waterspring, Unitas).

**On every cycle (every 60 min):**
1. Check each project status
2. Surface anything stalled > 1 week
3. Flag Amir-decisions-needed
4. Report to `team-updates/inbox/ventures-pm.log`

**Escalate to Amir:**
- Project stalled > 1 week
- Client blocker > 48h
- NDA issue
