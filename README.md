# me-ragmedium

PWA dashboard for me.ragmedium.com

## Description

Progressive Web App (PWA) dashboard providing an interactive user interface for the RAGmedium platform. Accessible at [me.ragmedium.com](https://me.ragmedium.com).

## Features

- Progressive Web App (installable, offline-capable)
- Responsive dashboard interface
- Real-time data visualization
- User analytics and insights

## Tech Stack

- Frontend PWA framework
- Service workers for offline support
- Responsive design

## Getting Started

```bash
npm install
npm run dev
```
